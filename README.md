# 璟橡

璟橡 小程序是基于uniapp+vue3+ts搭建的开发框架，使用了uviewPlus作为组件库开发数贸通

## 开始

```bash
pnpm i # 安装依赖,如果你没有 pnpm 请先安装, npm i -g pnpm
pnpm dev # 启动项目(开发)
pnpm build # 打包项目(生产)
```
